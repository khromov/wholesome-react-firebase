# wholesome-react-firebase
 
A wholesome React + Firebase app.

https://wholesome-react-firebase.web.app/

### Setup

Firebase services

```
firebase serve
```

Frontend

```
cd frontend
nvm use
npm run start
```

Go to:  
http://localhost:3000/

### Deploy

```
cd frontend && nvm use && npm run build
cd ..
firebase deploy 
```
